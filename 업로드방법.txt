GitHub 업로드 (폴더째 드래그!)
1. zip 압축해제 → api/, public/ 폴더 확인
2. GitHub 레포(unisquads) 기존 파일 전부 삭제
3. Add file → Upload files → api·public 폴더 통째로 드래그
4. Commit → 자동 재배포

★ 환경변수: JUSO_CONFM_KEY, KAKAO_REST_KEY, VWORLD_API_KEY

이번 기능 (IROS 고장 감지 배너):
- 등기조회가 구조이상(파싱/HTTP/세션/요청제한)으로 연속 3회 실패하면
- 화면 상단에 빨간 경고 배너 자동 표시
- "IROS 응답 구조가 바뀌었을 가능성 (파서 점검 권장)"
- 정상 실패(검색결과없음)나 일시적 실패는 배너 안 뜸
- 설정 불필요 (이메일·SMTP·서버 없음)
