GitHub 업로드 (폴더째 드래그!)
1. zip 압축해제 → api/, public/ 폴더 확인
2. GitHub 레포(unisquads) 기존 파일 전부 삭제
3. Add file → Upload files → api·public 폴더 통째로 드래그
4. Commit → 자동 재배포

★★ 이번 수정 ★★
1) 아파트면 동·호 판이 자동으로 깔림 (버튼 클릭 불필요)
   - 정제 → 집합건물 확정 → 즉시 [101동][102동] 판 표시
   - 후보 클릭으로 아파트 확정돼도 자동
2) 동·호 유실 버그 수정
   - 건물명만 정제해도 입력칸의 동·호가 IROS 검색어에 포함됨
3) 집합건물인데 동·호 없으면 조회 막고 입력 유도
