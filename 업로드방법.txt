GitHub 업로드 (폴더째 드래그!)
1. zip 압축해제 → api/, public/ 폴더 확인
2. GitHub 레포(unisquads) 기존 파일 전부 삭제
3. Add file → Upload files → api·public 폴더 통째로 드래그
4. Commit → 자동 재배포

이번 수정 (후보 클릭 + 동호 판):
- 후보 클릭하면 재검색 없이 바로 확정 (실패 버그 제거)
- 후보 클릭 시 그 아파트가 집합건물이면 동·호 토글 자동 열림
- isJip 판정을 juso 공동주택코드(bdKdcd) 기반으로 → 건물명에 키워드 없어도(삼환로즈빌 등) 정확 감지
