GitHub 업로드 (폴더 구조 유지 필수!)
1. zip 압축 해제 → addr-refine-deploy 폴더 (안에 api/, public/)
2. GitHub 레포(unisquads) 기존 파일 전부 삭제
3. Add file → Upload files → api 폴더, public 폴더를 통째로 드래그
4. Commit → Vercel 자동 재배포
※ 파일 하나씩 X, 폴더째 O
