GitHub 업로드 (폴더째 드래그!)
1. zip 압축해제 → api/, public/ 폴더 확인
2. GitHub 레포(unisquads) 기존 파일 전부 삭제
3. Add file → Upload files → api·public 폴더 통째로 드래그
4. Commit → 자동 재배포

★★ 이번 수정 (상태 오염 버그) ★★
- [버그] 아파트 검색 후 다른 주소(토지·단독) 검색하면
        이전 동·호(101/1502)가 그대로 남아 검색어를 오염시킴
        → "원주시 중앙동 50 101동 1502호" 같은 엉뚱한 검색어
- [수정] 정제 시 이전 세대판·에러·등기결과 초기화
        집합건물이 아니면 동·호 자동 제거
        집합건물일 때만 VWorld 호출 (토지·단독은 호출 안 함)
- 세대 버튼도 집합건물일 때만 표시

★ VWorld 도메인 등록 필수 (2곳 다):
  1) vworld.kr 인증키 관리 → 서비스URL에 배포주소 등록
  2) Vercel 환경변수 VWORLD_DOMAIN = 같은 URL
