IROS 진단 일괄

정제결과 엑셀을 올리면 각 주소를 실제 등기소(IROS)에 조회해 검색어 · 응답건수 · 실패원인을 한눈에 봅니다. 오류 원인 파악 전용 — 앱 동작에는 영향 없습니다.

동시 조회 수 · 최대 조회 건수

IROS는 건당 몇 초 걸리고 과도한 요청 시 차단될 수 있어, 소량(30건 내외)부터 권장합니다. 동시 조회 수를 올리면 빠르지만 차단 위험이 커집니다.

API 주소: https://unisquads.vercel.app · 이 페이지는 같은 도메인(unisquads.vercel.app/diag.html)에 올려서 여세요. 로컬 파일로 열면 CORS로 막힙니다.
